### 種子六義 → Atom 屬性

| 種子義 | Atom 對應 |
|--------|----------|
| 剎那滅（瞬間生滅） | Atom 每次載入都是新的 instance |
| 果俱有（與果同時） | Atom 載入即影響當前處理 |
| 恆隨轉（持續流轉） | Atom 隨 session 持續更新 |
| 性決定（善惡決定） | Confidence 等級（[固]/[觀]/[臨]） |
| 待眾緣（需外緣觸發） | Trigger 欄位（按關鍵詞比對載入） |
| 引自果（各引各果） | Scope 欄位（global/channel/person） |

## 他空見 → 擴增型輔助

| 概念 | 系統解讀 |
|------|---------|
| **不空**（本具佛性） | 本地記憶系統、決策模式、經驗累積 — 系統自身的核心能力 |
| **空**（客塵遍計） | 對外部 AI 的過度依賴、臨時性 context、不必要的複雜度 |
| **修行方向** | 逐漸強化本地能力，減少對 remote AI 的依賴 |
| **光明識流** | 系統的本質目標：擴增型輔助，而非代工型替代 |
